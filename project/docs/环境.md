环境
操作系统
系统：Windows
版本：Windows 10/11（需开启 WSL2 以支持 Docker 高性能运行）
Python 环境
Python 版本：3.10+（推荐 3.11，完美兼容 FastAPI 异步特性）
包管理工具：uv（推荐）或 pip
核心依赖：fastapi, uvicorn, requests, xmltodict, python-dotenv
IDE
PyCharm 版本：2023.3.5（或更高版本）
配置：建议创建独立虚拟环境，配置 Python 解释器，开启代码自动格式化与类型检查
Docker（必需）
用途：本地一键部署 Dify 平台（含 API 网关、PostgreSQL、Redis 等组件）
下载地址：https://www.docker.com/products/docker-desktop/
运行要求：确保 Docker Desktop 后台常驻，WSL2 分配内存 ≥4GB，终端可正常执行 docker compose
微信公众号（必需）
账号类型：已认证的服务号（订阅号无客服消息接口权限）
配置项：需准备 AppID、AppSecret、自定义 Token、EncodingAESKey
权限要求：后台需开启“服务器配置”与“客服消息”接口权限
域名与网络（必需）
域名要求：已完成 ICP 备案的公网域名（微信强制校验）
HTTPS 证书：需配置有效 SSL 证书（Let's Encrypt 或云厂商免费证书）
反向代理：Nginx（用于将 https://你的域名/wechat 代理至本地 127.0.0.1:8000）
本地穿透（可选）：Ngrok / Cpolar（无公网 IP 时用于临时生成 HTTPS 测试地址）
测试与调试工具
API 验证：Postman 或 PowerShell curl.exe（测试 Dify /v1/chat-messages 接口）
日志监控：PyCharm 终端或 CMD 实时查看 uvicorn 运行日志，配合 logger 输出追踪消息链
